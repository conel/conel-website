Bigmovie

This element facilitates the upload of a flv file and the output of the flv file in the document.

Please note:

The player element needs to be provided by in the template. This datatype only facilitates the upload and storage of the movie file.

Parameters

width

width="440"

Defines the width of the output.

height

height="300"

Defines the height of the output.
