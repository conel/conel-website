Listview

Reads pages with a certain template and formats the results with a definable output template. Different filters (see: field, category) can be used to define which pages will be shown.

Parameters

field

field="HEADLINE equal Searchword"

With this attribute you can search for all pages which contain a textfiled that contains a certain value. You can use either field or category.

  • equal
  • notequal
  • contains

category

category="basename"

If you want to show pages with a a specific template you should use category. You can include several basenames / templates. They should be listed seperated by comma.

scope

scope="local"

You should define this attibute in conjunction with category. Scope defines in which radius the search takes place. Possible values for scope are:

  • page
    Searches in all pages which are located a level below the page which is defined in pagename. For example: scope=“page” pagename=“homepage”
  • level
    Searches all pages which are on the same level as the current page.
  • local
    Searches all pages which are on the level below the current page.
  • branch
    Searches all pages below the current level.
  • global
    Searches all pages.

count

count="1"

This attribute defines how many found pages should be shown.

template

template="example"

This attribute defines the template which should be used to show the search results.

highlight

highlight="true"

This attribute defines if the current page should be highlighted in the output. See active section of the template description.

sortby

sortby="date"

This attribute defines which field should be used to order the results. Sortby is only available when the scope is global or local.

orderdirection

orderdirection="asc"

With this attribute you can define if the search results should be shown in ascending or descending order.

  • ASC
    Ascending search
  • DESC
    Descending search

sort

sort="date"

This attribute defines which algorithm will be used to perferm the search.

  • TEXT
    Alphanummeric search
  • DATE
    Chronological search

Output

body

< !-- BEGIN body -- >...< !-- END body -- >

This section is returned for each result.

head

< !-- BEGIN head -- >...< !-- END head -- >

If the search returns result the section head will be output before any results.

foot

< !-- BEGIN foot -- >...< !-- END foot -- >

If the search returns result the section foot will be output after any results.

empty

< !-- BEGIN empty -- >... < !-- END empty -- >

If no results are found this section will be output.

active

< !-- BEGIN active -- >...< !-- END active -- >

If you set the attribute highlight to true the current page will be output using this section.

alternate

< !-- BEGIN alternate -- >...< !-- END alternate -- >

If you want to have different ouputs for even and odd results this section will output a variation of the body section for odd results.

Variables

matrix:TARGETPAGE

Returns the URL of the target page

matrix:PAGEID

Returns the page id of the target page.

matrix:THISPAGE

Returns the ID number of the page which contains the listview.

matrix:COUNTER

Outputs the internal counter which starts with 0

matrix:ID

Outputs the internal counter which starts with 1.

matrix:PAGENAME

Returns the name of the target page.

matrix:THISPAGEURL

Returns the URL of the page the listview is on
