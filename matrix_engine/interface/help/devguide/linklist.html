Linklist

Allows the linking of internal pages. This element is repeatable and allows an unlimited number of links in one list.

Parameters

maxcount

maxcount="3"

This attribute defines the maximum number of links which can be made by the user.

template

template="basename"

This attribute defines which template is used to format the links.

Output

body

< !-- BEGIN body -- >...< !-- END body -- >

This section outputs the main part of the element.

head

< !-- BEGIN head -- >...< !-- END head -- >

If elements are found this section is ouput before of the individual body sections.

foot

< !-- BEGIN foot -- >...< !-- END foot -- >

If elements are found this section is ouput after the output of the individual body sections.

empty

< !-- BEGIN empty -- >...< !-- END empty -- >

If no elements are found the empty section is output.

Variables

matrix:TARGETPAGE

Outputs the URL of the target page.

matrix:PAGENAME

Outputs the name of the target page.
