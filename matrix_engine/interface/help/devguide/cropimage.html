Cropimage

This datatype provides a cropping functionality after the upload of an image. The cropped file is then saved and output.

Parameters

width

width="400"

Defines which width the cropped image should be. Images that are uploaded with a smaller width will be rejected.

height

height="50"

Defines which height the cropped image should be. Width and height can be used together or individually.

showalt

showalt="TRUE"

Enables an alt tag input box.

thumb_width

thumb_width="200,100"

Needs to be used in conjunction with thumb_height. Each entry in thumb_width / thumb_height prompts another cropping process. This way the user can be asked to provide several crops of the same image.

thumb_height

thumb_height="123,50"

See thumb_width.

Variables

identifier.width

Outputs the width of the image.

identifier.height

Outputs the height of the image.

identifier.file

Outputs a full path to the location of the image file and the file name.

identifier.filename

Outputs the name of the file.

identifier

Outputs a complete HTML-tag img for the image.

identifier.absolute

Outputs to absolute path to the image

identifier.X

Outputs the thumbnail X. The thumbnails are numbered from 1 depending on how many have been defined in the thumb_width / thumb_height parameters.
