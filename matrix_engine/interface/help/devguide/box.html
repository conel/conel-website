Box

Provides a "blackbox" for other datatypes. A box can hold an unlimited combination of different datatypes as a list.

Parameters

template

template="moreinfo,headline"

Defines a list of selectable templates in a comma seperated list. There should be no spaces between the entries and the commas.

entry

entry="TEXT:title,TEXT:copy,TEXT:date"

This attribute defines which element of the user selected template should be shown in the edit mode as information. The attribute can contain several elements as a comma seperated list. Again this list should not contain any spaces.

The format of an element entry is:
DATATYPE:name

maxcount

maxcount="5"

This attribute defines the maximum number of elements which can be added to the box.

boxtemplate

boxtemplate="basename"

This parameter is optional. The box template should contain the head, foot, empty, navigation blocks.

slideshow

slideshow="true"

If set the box will switch to slideshow mode. You should provide a boxtemplate. If set identifier.next and identifier.previous will be output.

Output

head

< !-- BEGIN head -- >...< !-- END head -- >

The head element will only be output when boxtemplate is set and it is defined in the template. It will be output before all other elements.

foot

< !-- BEGIN foot -- >...< !-- END foot -- >

will only be output if boxtemplate is set. Will be output qat the end of all box elements.

empty

< !-- BEGIN empty -- >...< !-- END empty -- >

Requires boxtemplate to be set. Will be output if no box elements where created.

navigation

< !-- BEGIN navigation -- >...< !-- END navigation -- >

Requires boxtemplate to be set. Will be output if slideshow is set to true.

Variables

identifier

Outputs the list of datatypes.

identifier.next

Outputs the link to the next box.

identifier.previous

outputs the link to the previous box.
