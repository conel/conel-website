Image

Allows the addition of images. If a size has been defined the uploaded image will be scaled to this size. If only one dimension (width or height) has been defined the image will be scaled in the correct aspect ratio.

Parameters

width

width="171"

Width of the image.

height

height="100"

Height of the image.

showalt

showalt="true"

If set to true a text input field for entering ALT text for an image will be displayed.

Variables

identifier.width

Outputs the width of the image.

identifier.height

Outputs the height of the image.

identifier.file

Outputs a full path to the location of the image file and the file name.

identifier.filename

Outputs the name of the file.

identifier

Outputs a complete HTML-tag img for the image.

identifier.absolute

Outputs to absolute path to the image

identifier.alt

Outputs the contents of the alt tag field.
