Copytext

This element is used for multiline text. The text can contain HTML tags.

Parameters

columns

columns="50"

Length of the text entry field in the edit mode.

lines

lines="10"

Hight of the text input field in the edit mode.

maxchar

maxchar="120"

Limits the number of allowed characters and outputs a counter which shows how many characters remain.
