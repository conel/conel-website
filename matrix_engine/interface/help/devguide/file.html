File

This elements is used to provide a file upload facility. The user can also specify a name for his upload.

Variables

identifier.text

Outputs the name which was entered by the user.

identifier.file

Outputs a full path to the uploaded file.

identifier.filename

Outputs the real file name.

identifier.size

Outputs the size of the file.

identifier

Outputs a complete link to the file.

identifier.absolute

Ouputs a complete absolute link to the file.

identifier.absolutefile

Outputs the absolute path and file name.
