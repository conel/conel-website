Email

Allows the user to enter a link text and an email. Both elements will be uft-8 entity encoded which protects from a majority of spam harvesting bots.

Variables

identifier

Outputs an complete mailto link.

identifier.link

Outputs the email address.

identifier.name

Outputs the name of the email link.
