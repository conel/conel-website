Include

Includes an external function from the file user_modules.php.

Parameters

function

function="myFunction"

Defines the name of the function which is called.
The structure_id of the calling page is passed automatically as first argument. The second argument is an associative array.

myFunction($structure_id,$parameter) $parameter = array('page_id' => {page_id of the page})

parameter

parameter="myVariable"

Defines a variable which is added to a function call. Is the variable set then the argument $parameter contains the defined value.

$parameter = array('page_id' => {page_id of the page}, 'parameter' => {defined value}

Variables

identifier

Outputs the returned value of the called function.
If the function returns a associative array then the keys well be used as variables instead of the identifier.
