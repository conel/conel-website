Date

Outputs a formated date.

Parameters

format

format="d.m Y"

Defines the output format of the date. The syntax is the same as for the PHP date() function.

The default date format set in the config.php file will be used whenever the format parameter is not set.

time

time="true"

Allows you to enter the time.

Variables

identifier

Outputs the formated date.

identifier.relative

Outputs the following:
If the date element is 1 or more months ago or in the future it will be output as specified on the format paramater (or the default settings in settings.php)


If there are 1 or more months, return just a date.

If there are 1 or more weeks, return weeks and, if necessary, days.

If there are 1 or more days, return days and, if necessary, hours.

If there are 1 or more hours, return hours and, if necessary, minutes.

If there are 1 or more minutes, return minutes.

Just return seconds.

If the date is in the past “ago” will be added to the date- otherwise “in”.

E.g.: 1 day and 16 hours ago
or in 1 day and 16 hours
