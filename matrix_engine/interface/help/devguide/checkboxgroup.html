checkboxgroup

Displays a number of checkboxes for the user to select.

Parameters

lables

labels="future,eco,branding"

This attribute defines the lables for the checkboxes as they are shown in the edit mode and output in the document.

options

options="1,2,3"

This attribute should have the same number of entries as the labels attribute. It defines the actual values which are saved in the database.

itemcount

itemcount="3"

This attribute defines how many checkboxes are shown per line in the edit mode.
